# Rarotonga Medical Centre Website

This website was developed as part of my NCEA Level 3 Digital Technologies assessment.

The project involved designing and developing a responsive website for a local medical and dental clinic using:

- HTML
- CSS
- JavaScript

Features:
- Responsive layout
- Structured navigation
- Service information pages
- Contact/appointment form
- Clean and accessible UI design

This project demonstrates front-end development skills, project planning, and real-world problem solving.
